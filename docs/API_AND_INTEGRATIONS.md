# API and Integrations Roadmap

## Phase 1 — Manual MVP
- Manual monthly metrics input.
- Manual project/task management.
- Manual website/SEO/GBP deliverable upload links.

## Phase 2 — Marketing integrations
- Google Search Console API for organic clicks, impressions, CTR, average position.
- Google Analytics 4 API for sessions, conversions, traffic source.
- Google Business Profile API for calls, direction requests, profile views if approved and available.
- CallRail or WhatConverts for call tracking.

## Phase 3 — Billing and operations
- Stripe subscription billing.
- QuickBooks invoices.
- Zapier/Make automation for lead routing.

## Phase 4 — White-label delivery controls
- Contractor workspaces.
- Activity logs.
- Client approval gates.
- QA checklist before publishing.
